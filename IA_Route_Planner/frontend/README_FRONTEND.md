# Frontend IA Route Planner

Este es el frontend de la aplicación IA Route Planner, construido con React y Vite, estilizado con Tailwind CSS.

## Instalación

1. Asegúrate de tener Node.js instalado (versión 16 o superior).
2. Navega al directorio frontend:
   ```
   cd frontend
   ```
3. Instala las dependencias:
   ```
   npm install
   ```

## Ejecución

1. Inicia el servidor de desarrollo:
   ```
   npm run dev
   ```
2. Abre tu navegador y ve a `http://localhost:5173` (o el puerto que indique Vite).

## Construcción para Producción

Para construir la aplicación para producción:
```
npm run build
```

Para previsualizar la build:
```
npm run preview
```

## Notas

- Asegúrate de que el backend esté corriendo en `http://localhost:5000` para que el frontend pueda hacer las peticiones.
- La aplicación es responsive y funciona en dispositivos móviles y de escritorio.
