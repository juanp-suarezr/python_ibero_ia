# Entregable Final IA Route Planner

## Resumen del Proyecto

El proyecto IA Route Planner es un sistema inteligente para calcular rutas óptimas en transporte masivo, implementando algoritmos de búsqueda informados y no informados, con una base de conocimiento en formato Prolog y un frontend responsive.

## Componentes Implementados

- **Backend**: API Flask con algoritmos Dijkstra, A*, BFS.
- **Frontend**: React + Tailwind CSS, responsive.
- **CLI**: Interfaz de línea de comandos.
- **Knowledge Base**: Parser estilo Prolog para hechos edge/3 y closed/1.
- **Documentación**: README, tests, guion de video.
